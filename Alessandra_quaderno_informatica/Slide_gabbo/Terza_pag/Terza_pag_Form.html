Illustrazione dell'Attività

Dettagli dell'attività:

    Questo compito gestisce i dati anagrafici degli utenti e li visualizza in una tabella HTML. 
    Quando l'utente invia il modulo con le sue informazioni, questi dati vengono recuperati, 
    puliti e successivamente mostrati in una tabella ben formattata.

Procedimento Logico:

    1. L'utente compila il modulo con i propri dati personali.
    2. Quando il modulo viene inviato, le informazioni vengono inviate tramite il metodo POST.
    3. Viene utilizzata una funzione per "pulire" i dati (rimuovendo spazi extra e prevenendo input dannosi tramite htmlspecialchars).
    4. I dati vengono visualizzati in una tabella HTML, dove ogni campo è mostrato con il rispettivo valore.
    5. Se non vengono inviati dati, viene visualizzato un messaggio di errore.

Snippet di Codice PHP Importante:

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    function pulisci($dato) {
        return htmlspecialchars(trim($dato));  // Rimuove spazi e impedisce XSS
    }

    $nome = pulisci($_POST['nome']);
    $cognome = pulisci($_POST['cognome']);
    $data_nascita = pulisci($_POST['data_nascita']);
    $email = pulisci($_POST['email']);
    $cellulare = pulisci($_POST['cellulare']);
    $via = pulisci($_POST['via']);
    $cap = pulisci($_POST['cap']);
    $comune = pulisci($_POST['comune']);
    $provincia = pulisci($_POST['provincia']);

    echo "table border='1' cellpadding='5'";
    echo "tr th Campo /th th Valore /th /tr";
    echo "tr td Nome /td td $nome/td /tr";
    echo "tr td Cognome /td td $cognome /td /tr";
    echo "tr td Data di nascita /td td $data_nascita /td /tr";
    echo "tr td Email /td td $email /td /tr";
    echo "tr td Cellulare /td td $cellulare /td /tr";
    echo "tr td Via/Piazza /td td $via /td /tr";
    echo "tr td CAP /td td $cap /td /tr";
    echo "tr td Comune /td td $comune /td /tr";
    echo "tr td Provincia /td td $provincia /td /tr";
    echo "table";
}

Comandi Importanti da Ricordare:

$_SERVER['REQUEST_METHOD']:
Cosa fa: Controlla il tipo di richiesta (POST o GET) inviata dal browser. In questo caso, verificamosi se è stato inviato un modulo con il metodo POST.

htmlspecialchars():
Cosa fa: Converte i caratteri speciali in entità HTML, prevenendo possibili attacchi di tipo XSS (Cross-Site Scripting) che potrebbero essere causati da input malevoli.

trim():
Cosa fa: Rimuove gli spazi vuoti all'inizio e alla fine di una stringa, garantendo che i dati siano puliti prima di essere utilizzati o visualizzati.
